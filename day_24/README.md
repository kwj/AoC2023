# Day 24

## Requirement

* [Julia](https://julialang.org/) (confirmed to work with Julia v1.9.4)

## How to use

### Read the source file in REPL

```console
julia> include("d24.jl")
```

### Run each solver with the name of the puzzle data file as an argument

*Part one*

```console
julia> d24_p1("input")
```

*Part two*

```console
julia> d24_p2("input")
```
