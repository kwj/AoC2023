# Solutions to Day 13

## Requirement

* [Julia](https://julialang.org/) (confirmed to work with Julia v1.9.4)

## How to use

### Read the source file in REPL

```console
julia> include("d13.jl")
```

### Run solver with the name of the puzzle data file as an argument

*Part one* and *Part two*

```console
julia> d13("input")
```
