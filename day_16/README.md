# Solutions to Day 13

## Requirement

* [Rust](https://www.rust-lang.org/) (confirmed to work with Rust 1.74.1)

## How to use

### Compile the source file

```console
$ rustc d16.rs
```

### Run solver with the name of the puzzle data file as an argument

*Part one* and *Part two*

```console
$ ./d16 input
```
