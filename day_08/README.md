# Solutions to Day 8

## Requirement

* [Deno](https://deno.com/) (confirmed to work with Deno v1.38.5)

> Note: This program may be easily ported to other runtimes, such as Node.js.

## How to use

*Part one*

```console
deno run --allow-read=./ d08.ts 1 input
```

*Part two*

```console
deno run --allow-read=./ d08.ts 2 input
```
